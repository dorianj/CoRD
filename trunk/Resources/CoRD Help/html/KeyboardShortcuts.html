CoRD Keyboard Shortcuts

CoRD has a number of keyboard shortcuts preassigned for rapid navigation and efficiency.

Hotkey Action
⌘NCreates a new Server
⌘⇧DToggle Servers Drawer
⌘WClose Current Session
⌘⌥WClose Current Window
⌘IToggle Server Inspector
⌘UToggle Unified/Windowed Mode
⌘↩Start Full Screen Mode
⌘GJump to Quick Connect
⌘⇧GJump to Search Servers
⌘⌥1Brings main CoRD window to the front
⌘}Selects Next Active Session
⌘{Select Previous Active Session
⌘1 thru ⌘9User-Definable, can bind saved servers to these keys
