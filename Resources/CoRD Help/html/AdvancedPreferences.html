CoRD Help

Advanced Preferences

  • Use SOCKS Proxy Forces CoRD to use the defined SOCKS proxy defined in Network Preferences.
  • Ignore Installed Location CoRD has built-in location checking to make sure its located in an Applications folder. If you want CoRD to live in a non-recognized Applications folder, this allows you to ignore those prompts.
  • Logging Logs activity to ~/Library/Logs/CoRD.log.

Screen Resolutions

Allows a custom list of resolutions to be defined. Resolutions must be entered in the proper format, ie. 0000x0000.
